---

## Ops

Operator-style functions.

---

### `yes(value)`

Same as `!!`.

```js
yes(null)
// false

yes(1)
// true
```

---

### `no(value)`

Same as `!`.

```js
no(null)
// true

no(1)
// false
```

---

### `add(a, b)`

Same as `+`.

```js
add(1, 2)
// 3
```

---

### `sub(a, b)`

Same as `-`.

```js
sub(2, 1)
// 1
```

---

### `inc(num)`

Increments by `1`.

```js
inc(1)
// 2
```

---

### `dec(num)`

Decrements by `1`.

```js
dec(2)
// 1
```
