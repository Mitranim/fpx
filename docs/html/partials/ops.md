## Ops

Operator-style functions. Sometimes useful with higher-order functions.

---

### `add(a, b)`

Same as `+`.

```js
add(1, 2)
// 1 + 2 = 3
```

---

### `sub(a, b)`

Same as `-`.

```js
sub(2, 1)
// 2 - 1 = 1
```

---

### `mul(a, b)`

Same as `*`.

```js
mul(10, 20)
// 10 * 20 = 200
```

---

### `div(a, b)`

Same as `/`.

```js
div(10, 20)
// 10 / 20 = 0.5
```

---

### `inc(num)`

Increments by `1`.

```js
inc(1)
// 1 + 1 = 2
```

---

### `dec(num)`

Decrements by `1`.

```js
dec(2)
// 2 - 1 = 1
```
